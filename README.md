## Getting Started

### Prerequisites

* A raspberry pi
* python3
* pip3
* A Pimoroni Inky Impression 

### Installation

1. Clone the repo to your pi
2. Setup a virtual env
```sh
python3 -m venv venv
```
3. Activate it
```sh
. venv/bin/activate
```
4. Install dependencies
```sh
pip install -r requirements.txt
```
5. Run the flask server
```sh
flask run
```
